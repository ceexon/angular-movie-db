# angular-movie-db

Installation
---
1. Clone the Repository

        $ git clone git@github.com:kburudi/angular-movie-db.git
        
2. checkout the directory and Install Dependencies

        $ cd angular-movie-db

        $ yarn || npm install
        
3. Create a .env and add your moviedb API Key.

4. start the server

        $ ng serve
